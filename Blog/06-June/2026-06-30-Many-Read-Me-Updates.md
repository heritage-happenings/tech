Welcome new visitors!

In the tech world, when you build software and put it on the web, you almost always add a README file.

The [README file](https://en.wikipedia.org/wiki/README) is a simple text file that explains what the software does, how to use it, and any other important information. It is often the first thing that users see when they visit a software project, and it can be a valuable resource for understanding the project and getting started with it.

Me being me, I like to include README files most anywhere they could be.

So one of the tricks of this website is to automatically open the README when you open up a folder.

And I can do this now very easily because AI is helping me update thefiles. And there were many updates today. ;-)

Nonetheless, AI has errors and hallucinations. I've checked a few of the things, but not all, so please do report any inconsistencies.
