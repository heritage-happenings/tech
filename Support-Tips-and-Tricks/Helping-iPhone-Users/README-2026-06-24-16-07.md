# Helping iPhone Users

The following infographics are designed to provide you with an overview of how to assist iPhone users more effectively - no matter their level of experience or familiarity with the device.

They are presented as individual tips.

Each tip has an infographic that is designed to be read by and kept by residents.

Each tip also has a poster that can be used by team members

All of these tips have been freshly created using the help of Claude AI. Therefore, they can all be debugged, improved, added to, changed, or whatever.

Please do let Theo or Claire know if you have any suggestions for improvements, or if you would like to add your own tips to this collection.
