## Key Features to Focus On First
When helping a resident configure their device, changing everything at once can be overwhelming. These three high-impact settings usually offer the most immediate relief:
| Feature | What It Does | Why It Helps |
|---------|-------------|-------------|
| Easy Mode / Simple Launcher | Replaces standard layout with massive icons and a basic grid. | Prevents accidental swipes and reduces visual clutter. |
| Display & Font Size | Globally scales up text and menu elements across the OS. | Eliminates squinting and makes buttons easier to target accurately. |
| 3-Button Navigation | Swaps modern swipe gestures for three clear buttons at the bottom. | Provides a more familiar and accessible navigation method. |

## Simple Tricks for Missed Calls
- Like many people I keep my phone
ringer on vibrate, but I don’t usually carry my phone on me – I may leave it on a desk – so I often miss calls. I’ve greatly reduced missed calls by setting the phone to flash its flashlight and flash its screen while it vibrates. That flashing light is enough to notice from a distance.

It is easy to program on the iPhone. Go Settings > Accessibility > Audio Visual > Flash for Alerts.

For Android: Settings > Accessibility > Audio & Screen Text > Flash Notifications. KK

## [All Seniors Foundation Guide to Smartphone Setup]( https://allseniors.org/articles/how-to-set-up-a-senior-friendly-smartphone/ )

This foundation provides an exceptionally practical, empathetic approach. Instead of just listing features, their guide focuses on creating a "calm" phone environment. It covers how to properly scale text, clean up the home screen, and set up simple contact shortcuts using plain language.
